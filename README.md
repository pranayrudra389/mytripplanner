# MyTripPlanner

Repository scaffold for MyTripPlanner with:

- `apps/web` for the frontend
- `apps/api` for API routes and controllers
- `packages/agent-core` for shared AI orchestration logic
- `packages/plugins/travel` for travel-specific workflows and tools
- `database` for migrations and seed data
- `docs` for product and architecture documentation

The existing Python prototype remains in the repository root.
